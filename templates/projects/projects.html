Role-Based Access Rights Model for Cloud System    (Dec 2013 – Apr 2014)

  • Role-based access rights provide a hierarchical model to read and write the data as per authorization and requirement of the implementation logic. The generic models have security drawbacks and are vulnerable to unethical exploits.
  • The proposed model uses broadcast encryption technique and decrypts using respective key. This scheme guarantees, other users/roles are not affected when revoking a user, re-encryption is not needed after user revocation.
  • Published paper in ARPN journals on final project report, 2014.
